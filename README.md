# ChorePal – Smart Chore Management Android App

## 📌 Project Overview
ChorePal is a mobile application designed to help parents manage household chores efficiently while encouraging children to build responsibility through a structured reward system.

The app allows parents to assign tasks, monitor progress, and approve completed chores, while children can view their assigned chores, submit proof of completion, and track their earned rewards.

This project is being developed as part of a Capstone course and focuses on building a real-world, scalable mobile application using modern software engineering practices.

---

## 🎯 Objectives
- Simplify household task management for families
- Encourage responsibility and discipline in children
- Provide a structured reward system
- Build a user-friendly Android application
- Apply full software development lifecycle (SDLC)

---

## 🚀 Key Features

### 👨‍👩‍👧 Parent Features
- Create and assign chores
- Edit or delete chores
- Approve or reject completed tasks
- Track children’s progress and rewards
- Manage household activities

### 👦 Child Features
- View assigned chores
- Mark chores as completed
- Upload proof (photo-based verification – future feature)
- Track earned rewards

### 🔐 Authentication
- Secure login system
- Role-based access (Parent / Child)

### 🔔 Additional Features
- Notification system (basic)
- Reward tracking system
- Dashboard for both parent and child

---

## 🏗️ System Architecture
The application follows a client-server architecture:

- **Frontend:** Android Mobile App (Kotlin)
- **Backend:** RESTful API (planned)
- **Database:** MongoDB / Firebase (planned)
- **Authentication:** JWT-based system (planned)

---


## 📱 Platform

Future versions of the application may include iOS support as part of extended development.
